HTML Reference

HTML Formatting Elements

Tag Name Description
<b> Bold Text This formatting element causes text to appear bold
<strong> Important Text This formatting element causes text to appear bold
<i> Italic Text This formatting element causes text to appear italic
<em> Emphasized Text This formatting element causes text to appear italic
<mark> Marked Text This formatting element causes the text to be highlighted
<small> Smaller Text This formatting element causes the text to appear smaller
<del> Deleted Text This formatting element causes the text to have a line through it
<ins> Inserted Text This formatting element causes the text to have a line underneath
<sub> Subscript Text This formatting element causes the text to appear smaller and lower then the baseline
<sup> Superscript Text This formatting element causes the to appear smaller and higher then the baseline

HTML Quotation & Citation Elements

Tag Name Description
<blockquote> Block Qoute This Quotation element defines a section that is quoted from another source
<q> Qoute This Quotation element defines a short quotation
<abbr> Abbreviations This Citation element tag defines an abbreviation or an acronym
<address> Adress This Citation element tag defines the contact information for the author/owner of a document or an article. It can also be an email address, url, physical address, phone number, social media handle, etc.
<cite> Citation This Citation element tag defines the title of a creative work (e.g a book, poem, song, movie, painting, etc.)
